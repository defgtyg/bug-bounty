# Signal in the Audit Log

In the quiet pane where build lights fade,
A warning hums beneath the green;
Not every fault arrives displayed,
Some wait between the lines unseen.

A token sleeps in borrowed shade,
A route accepts what none should send;
We trace the branch the shortcut made,
Then teach the gate where trust must end.

The patch is small, the proof is plain,
A failing path now turns aside;
The test repeats the former pain
And keeps the fix awake inside.

So ship the diff with measured care,
Let changelogs note the risk made known;
Security is work we share,
A watch kept best by more than one.
